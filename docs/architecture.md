# 架构设计

## 总工程结构

```
Frame_Me/
├── CLAUDE.md              # 总工程入口
├── docs/                  # 总工程知识库
├── frame-me-parent/       # 脚手架父工程
│   ├── CLAUDE.md
│   └── docs/
└── fm-demo/               # 演示聚合服务
    ├── CLAUDE.md
    └── docs/
```

## 分层原则

- 每个子工程独立演进，独立构建、测试、部署。
- 子工程之间通过接口契约、事件或 RPC 协作，不直接依赖彼此内部实现。
- 总工程提供跨工程约定索引，不承载业务代码。
